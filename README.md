# Ion's automation #

Welcome to Ion's automation test using Cypress.

### How do I get set up? ###

#### Requisites

* [NodeJS](https://nodejs.org)
* [NPM](https://npmjs.com)

#### Running locally 

1. Clone the project
  ```bash
  git clone git@bitbucket.org:scribblelive/ion-automation-cypress.git
  ```

2. Install dependencies
  ```bash
  npm install`
  ```

3. Run the tests
  ```bash
  npm run e2e
  ```
